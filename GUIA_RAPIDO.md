# Guia Rápido - Sistema Autônomo Hospitalar Saúde

Este guia foi criado para que qualquer pessoa possa entender o projeto em 5 minutos.

---

## O Que É Este Projeto?

Estamos criando **assistentes virtuais inteligentes** para cada setor da Hospitalar Saúde. Esses assistentes usam Inteligência Artificial para automatizar tarefas repetitivas, responder perguntas e ajudar os colaboradores a serem mais produtivos.

---

## Por Que Estamos Fazendo Isso?

| Antes (Problema) | Depois (Solução) |
|---|---|
| Orçamentos demoram horas para fazer | IA sugere itens automaticamente |
| Muitos erros em cálculos | IA valida tudo antes de finalizar |
| Dúvidas ficam sem resposta | IA responde 24 horas por dia |
| Relatórios manuais | IA gera relatórios automaticamente |

---

## Como Funciona?

```
┌─────────────────────────────────────────────┐
│                                             │
│   COLABORADOR  ──────►  AGENTE IA           │
│                              │              │
│                              ▼              │
│                    SISTEMA HOSPITALAR       │
│                              │              │
│                              ▼              │
│                    RESULTADO AUTOMÁTICO     │
│                                             │
└─────────────────────────────────────────────┘
```

1. O colaborador faz uma pergunta ou solicita uma tarefa
2. O agente de IA processa a solicitação
3. O agente consulta o sistema da empresa
4. O resultado é entregue automaticamente

---

## Quais São as Regras Principais?

### As 3 Regras Mais Importantes

1. **Tudo em Docker** - O código roda em containers isolados
2. **Foco em ROI** - Só desenvolvemos o que gera valor
3. **Documentação** - Tudo é registrado para não perder conhecimento

---

## Qual é o Primeiro Passo?

Estamos começando pelo **Setor de Orçamentos** porque:
- É um setor crítico para a receita
- Tem processos bem definidos
- Permite medir resultados facilmente
